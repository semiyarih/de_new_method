Differential Equation (DE)

\[ y'(t) = y(t) - y^2(t) ,\qquad \text{Logistic Growth}\]

\[y = \dfrac{k e^t} {(1 + ke^t)}\]

\[ \left(\dfrac{k e^t} {(1 + ke^t)}\right)' = \left(\dfrac{k e^t} {(1 + ke^t)}\right) + ke^t \left(\dfrac{-k e^t} {(1 + ke^t)^2}\right)\] \[ \left(\dfrac{k e^t} {(1 + ke^t)}\right)' = \left(\dfrac{k e^t} {(1 + ke^t)}\right) - \left(\dfrac{k e^t} {(1 + ke^t)}\right)^2\]

Initial Value Problems (IVPs) vs. Boundary Value Problems (BVPs)

  • Initial Value Problems (IVPs)

\[y'' + y + y^3 = 0, \qquad y(0) = 0,\; y'(0) = 2\] - Boundary Value Problems (BVPs)

\[y'' + y + y^3 = 0, \qquad y(0) = 0,\; y(1) = 1\]

IVP: \[ y' = y - y^2, \quad y(0)= 1/2\] Solution: \(\dfrac{e^t} {(1 + e^t)}\)

BVP: \[ y'' = y' - 2yy', \quad y(0)= 1/2, \; y(\ln(2)) = 2/3.\] Solution: \(\dfrac{e^t} {(1 + e^t)}\)

Linear vs. Nonlinear Differential Equations

  • Linear Differential Equations

\[y'' = x\, y, \qquad \text{Airy Equation}\] - Nonlinear Differential Equations

\[y'' + \delta y' + \alpha y + \beta y^3 = \gamma \cos(\omega t),\quad \text{Duffing Equation}\]

Types of Differential Equations

  • Ordinary Differential Equations (ODEs) \[y''(t) = -\sin(y(t))\]

  • Partial Differential Equations (PDEs) \[u_t(t,x) = u_{xx}(t,x) + u^2(t,x)\]

\(y'' = -y\)

\[ \begin{align*} \left\{ \begin{array}{c} y''=-y\\ y(0)=1,\qquad y(\pi/2)=1 \\[5mm] \end{array}\right. \end{align*} \]

The exact solution is \(y(t) = \cos t + \sin t\)

Define \(u = y'\), and

u' = y''
u' =  -y

\[ \begin{align*} \left\{ \begin{array}{c c} y'=u,&y(0)=1 \\[5mm] u'=-y\;,&u(0)=\gamma \end{array}\right. \end{align*} \]

The Integral Form Of The System

\[ \begin{cases} y(t) = \alpha + \int_0^t\, u(s) \, ds \\ u(t) = \gamma - \int_0^t\, y(s) \, ds \\ \end{cases} \]

Where \(\alpha = 1\), then

\[ \begin{cases} y(t) = 1 + \int_0^t\, u(s) \, ds \\ u(t) = \gamma - \int_0^t\, y(s) \, ds \\ \end{cases} \]

Setting Picard Iteration

We have derived the system of equations and now we want to set up Picard iterations.

Initial Values:

y(0) = 1

u(0) = y'(0)
y(0) = 1

u(0) = $\gamma$

\[ \begin{cases} y^{[0]} = \alpha = 1 \\ u^{[0]} = \gamma \\ \end{cases} \]

Picard Iteration Update Formulas \[ \begin{cases} y^{[k+1]}(t) = \alpha + \int_0^t\, u^{[k]}(s) \, ds \\ u^{[k+1]}(t) = \gamma - \int_0^t\, y^{[k]}(s) \, ds \\ \end{cases} \]

What to do with unknown \(\gamma\)?

Let’s find an initial guess!

Since \(\alpha= \beta = 1\), then

\[\gamma^{[0]} = \dfrac{\beta - \alpha}{b-a} =0 \]

Approximation for \(\gamma\):

To approximate \(\gamma\), use the integral form of the differential equation \(y'' = -y\) with \(y(0) = 1\).

\[y(t) = y(0) + y'(0)t - \int_0^t (t-s) y(s) \, ds\]

Given the boundary condition \(y(\pi/2) = 1\), and substituting \(t= \pi/2\), we obtain:

\[y(\pi/2) = 1 - \gamma \times \frac{\pi}{2} - \int_0^\frac{\pi}{2} (\frac{\pi}{2} - s) y(s) \, ds\]

Rearrange to solve for \(\gamma\):

\[\gamma =\frac{2}{\pi} \left( \int_0^\frac{\pi}{2} (\frac{\pi}{2} - s) y(s) \, ds\right)\]

Updated Picard Iteration with \(\gamma\):

\[ \begin{cases} y^{[k+1]}(t) = \alpha + \int_0^t, u^{[k]}(s) \, ds \\ u^{[k+1]}(t) = \gamma^{[k]} - \int_0^t, y^{[k]}(s) \, ds \\ \gamma^{[k+1]} =2/\pi( \int_0^{\pi/2} ((\pi/2) - s) y^{[k+1]}(s) \, ds) \end{cases} \]

minus_y <- function(n){
  library(pracma)
  n  # Number of iterations
  a <- 0
  alpha <- 1
  b<- pi/2
  beta <- 1
  
  h <- (b - a)
  # Set the Initial Conditions ---------------
  y <- c(0,alpha)
  u <- c(0, (beta - alpha)/h)
  g <- (beta - alpha)/h
  # Perform Picard iterations --------------
  for (k in 1:n) {
    # Compute y^[k+1]
    y_n <- anti_derivative(0, u)
    y_n[length(y_n)] <- alpha + y_n[length(y_n)] 

    # Compute u^[k+1]
    u_n <-  anti_derivative(0, -1*y)
    u_n[length(u_n)] <- g + u_n[length(u_n)] 
    
    # Compute g^[k+1] --------
    g <- gamma(a, b, alpha, beta, (-1*y))
    
    # update y and u
    y <- y_n
    u <- u_n
  }
  return(y)
}

     h=b-a;

\(y'' = -\sin y\)

\[ \begin{align*} \left\{ \begin{array}{c} y''=-\sin y \\ y(0)=1,\qquad y(1)=0 \\[5mm] \end{array}\right. \end{align*} \]

The system of the first order equations

\[ \begin{align*} \left\{ \begin{array}{c} y' = u\\ \\[2mm] u'=\sin y \\ \end{array}\right. \end{align*} \]

Define

\[ \begin{align*} \left\{ \begin{array}{c} u=y' \\ \\[1mm] v=\sin y \\ \\[1mm] w=\cos y \\ \end{array}\right. \end{align*} \] \[ \begin{align*} \left\{ \begin{array}{c} v'=y'\cos y \; =\; uw\ \\[2mm] w=-y'\sin y \; =\; -uv\ \end{array}\right. \end{align*} \]

The system of first-order

\[ \begin{align*} \left\{ \begin{array}{c c} y'=u,&y(0)=1 \\[5mm] v'=uw,&v(0)=\sin y(0) =\sin 1 \\[5mm] w'=-uv,&w(0)=\cos y(0) =\cos 1 \\[5mm] u'=-v,&u(0)=\gamma \end{array}\right. \end{align*} \]

Setting Picard Iteration

We have derived the system of equations and now we want to set up Picard iterations.

Initial Values: \[ \begin{cases} y^{[0]} = \alpha = 1 \\ u^{[0]} = \gamma \\ v^{[0]} = \sin \alpha = \sin 1\\ w^{[0]} = \cos \alpha = \cos 1 \end{cases} \]

Picard Iteration Update Formulas: \[ \begin{cases} y^{[k+1]}(t) = \alpha + \int_0^t\, u^{[k]}(s) \, ds \\ u^{[k+1]}(t) = \gamma - \int_0^t\, v^{[k]}(s) \, ds \\ v^{[k+1]}(t) = \sin 1 + \int_0^t\, u^{[k]} w^{[k]}(s) \, ds \\ w^{[k+1]}(t) = \cos 1 - \int_0^t\, u^{[k]} v^{[k]}(s) \, ds \\ \end{cases} \]

Let’s find an initial guess for \(\gamma\)

\[\gamma^{[0]} = \dfrac{\beta - \alpha}{b-a} = -1\]

Approximation for \(\gamma\):

To approximate \(\gamma\), use the integral form of the differential equation!

\[y(t) = y(0) + y'(0)t - \int_0^t (t-s) v(s) \, ds\]

Given the boundary condition \(y(1) = 0\), and substituting \(t= 1\), we obtain:

\[0 = 1 - \gamma \times 1 - \int_0^1 (1 - s) v(s) \, ds\]

Rearrange to solve for \(\gamma\):

\[\gamma = 1 + \int_0^1 (1 - s) v(s) \, ds\]

Updated Picard Iteration with \(\gamma\):

\[ \begin{cases} y^{[k+1]}(t) = \alpha + \int_0^t, u^{[k]}(s) \, ds \\ u^{[k+1]}(t) = \gamma^{[k]} - \int_0^t, v^{[k]}(s) \, ds \\ v^{[k+1]}(t) = \sin 1 + \int_0^t\, u^{[k]} w^{[k]}(s) \, ds \\ w^{[k+1]}(t) = \cos 1 - \int_0^t\, u^{[k]} v^{[k]}(s) \, ds \\ \gamma^{[k+1]} = 1 + \int_0^1 (1 - s) v^{[k+1]}(s) \, ds \end{cases} \]

siny <- function(n){
  library(pracma)
  n  # Number of iterations
  a <- 0
  alpha <- 1
  b<- 1
  beta <- 0
  
  h <- (b - a)
  # Set the Initial Conditions ---------------
  y <- c(0,alpha)
  v <- c(0, sin(alpha))
  w <- c(0, cos(alpha))
  u <- c(0, (beta - alpha)/h)
  g <- gamma(a, b, alpha, beta,v)
  # Perform Picard iterations --------------
  for (k in 1:n) {
    # Compute y^[k+1]
    y_n <- anti_derivative(0, u)
    y_n[length(y_n)] <- alpha + y_n[length(y_n)] 
    
    # Compute v^[k+1]
    v_n <- anti_derivative(0, cauchy_product(u, w))
    v_n[length(v_n)] <- sin(alpha) + v_n[length(v_n)]
    
    # Compute w^[k+1]
    w_n <-  anti_derivative(0, cauchy_product(u, -1*v))
    w_n[length(w_n)] <- cos(alpha) + w_n[length(w_n)]
    
    # Compute u^[k+1]
    u_n <-  anti_derivative(0, -1*v)
    u_n[length(u_n)] <- g + u_n[length(u_n)] 
    
    # Compute g^[k+1] --------
    g <- gamma(a, b, alpha, beta, (v))
    
    # update y and u
    y <- y_n
    v <- v_n
    w <- w_n
    u <- u_n
  }
  return(y)
}

\(y'' + 0.5y' - y + y^3 = 0\)

Let us try to solve the following boundary value problem (BVP), which is the Duffing equation:

\[y'' + 0.5y' - y + y^3 = 0, \quad y(0) = 0, \quad y(1/2) = 2.\]

We introduce the following substitutions:

\[ \begin{cases} u = y' \\ v = y^3 \\ w = y^2 \end{cases} \]

With these substitutions, we derive the following relations:

\[ \begin{cases} w' = 2y' = 2u \\ v' = 3y^2 = 3w\\ u' = -0.5u + y - v \end{cases} \]

Thus, the system of first-order differential equations becomes:

\[ \begin{cases} y' = u, \quad y(0) = 0 \\ u' = -0.5u + y - v, \quad u(0) = y'(0) = \gamma \\ w' = 2yu, \quad w(0) = y(0)^2 = 0 \\ v' = 3wu, \quad v(0) = y(0)^3 = 0 \end{cases} \]

Setting Picard Iteration

We have derived the system of equations and now we want to set up Picard iterations.

Initial Values: \[ \begin{cases} y^{[0]} = 0 \\ u^{[0]} = \gamma \\ v^{[0]} = 0 \\ w^{[0]} = 0 \end{cases} \]

Picard Iteration Update Formulas: \[ \begin{cases} y^{[k+1]}(t) = \int_0^t, u^{[k]}(s) \, ds \\ u^{[k+1]}(t) = \int_0^t, \left[ -0.5u^{[k]}(s) + y^{[k]}(s) - v^{[k]}(s) \right] \, ds \\ v^{[k+1]}(t) = 3 \int_0^t\, u^{[k]} w^{[k]}(s) \, ds \\ w^{[k+1]}(t) = 2 \int_0^t\, u^{[k]} y^{[k]}(s) \, ds \\ \end{cases} \]

\[\gamma^{[0]} = \dfrac{y(1) - y(0)}{ 1/2-0} = 4\]

Approximation for \(\gamma\): To approximate \(\gamma\), use the integral form of the differential equation!

\[y(t) = y(0) + y'(0)t + \int_0^t (t-s) \left(- 0.5y'(s) + y(s) - y(s)^3 \right) \, ds\]

Given the boundary condition \(y(\frac{1}{2}) = 2\), and substituting \(t= \frac{1}{2}\), we obtain:

\[2 = \gamma\times \frac{1}{2} + \int_0^{1/2} (\frac{1}{2} - s) \left( -0.5u(s) + y(s) - v(s) \right) \, ds\]

Rearrange to solve for \(\gamma\):

\[\gamma = 2\left(2 - \int_0^{\frac{1}{2}} (\frac{1}{2} - s) \left( -0.5u(s) + y(s) - v(s) \right) \, ds\right)\]

Updated Picard Iteration with ():

\[ \begin{cases} y^{[k+1]}(t) = \int_0^t u^{[k]}(s) \, ds \\ u^{[k+1]}(t) =\gamma^{[k]} - \int_0^t \left[ u^{[k]}(s) + y^{[k]}(s) + v^{[k]}(s) \right] \, ds \\ v^{[k+1]}(t) = 3 \int_0^t w^{[k]}(s) \, ds \\ w^{[k+1]}(t) = 2 \int_0^t v^{[k]}(s) \, ds \\ \gamma^{[k+1]} = 2(2 - \int_0^{1/2} (1/2 - s) \left( -0.5u^{[k]}(s) + y^{[k]}(s) - v^{[k]}(s) \right) \end{cases} \]
